Plugin - Stream Power Index (SPI) e Topographic Wetness Index (TWI) a partir de um Modelo Digital de Elevação (MDE/DEM), para QGIS / Projeto PRAD Toolbox (LabGEO UFR - LabGIS UFMS) 

Como instalar: a) abra o programa QGIS 3x; b) clique em "Complementos"; c) clique em "Gerenciar e Instalar Complementos"; d) clique em "Instalar a partir do ZIP"; e) clique em Procurar arquivo ZIP no computador ==> [...]; f) selecione o arquivo zip desejado; g) clique em "Instalar Complemento";
